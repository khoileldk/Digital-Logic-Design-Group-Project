# Digital Logic Design Group Projects using Quartus

This repository contains a collection of group projects developed for the Digital Logic Design course using Intel Quartus.

The projects focus on:

* FSM-based controller design
* Datapath architectures
* Sequential circuits
* Memory and register systems
* Pipelining techniques
* Hardware resource optimization

Implemented projects include:

* Shift Registers with Parallel Load
* BCD Counter
* Register File (2 Read Ports, 1 Write Port)
* RAM 32×32
* Stack and Queue
* Datapath and Controller systems
* Register Sharing
* Functional Unit Sharing
* Pipeline Datapath Architectures

> These projects were developed as part of a university group assignment.
> Reports and detailed explanations are written in Vietnamese.
